Lógica de Programação

O que são Linguagens de Programação? Fale sobre as seguintes linguagens: C++, Javascript, Java, Python, PHP.

Linguagens permitem escrever programas. C++: Orientada a objetos, eficiente. JavaScript: Para web, interpretada. Java: Plataforma independente. Python: Simples, versátil. PHP: Para web backend.

O que é uma linguagem de programação interpretada? e uma linguagem de programação compilada?

Interpretada: Executada linha por linha (ex: Python). Compilada: Convertida em código máquina antes (ex: C++).

O que é HTML e CSS

HTML estrutura páginas web. CSS estiliza.

O que é uma IDE (ambiente de desenvolvimento de sistemas - exemplos: VSCode, IntelliJ, …)

IDE facilita escrever código com editor, depurador, etc. Exemplos: VSCode, IntelliJ.
