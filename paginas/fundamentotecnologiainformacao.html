Principais componentes de hardware

Resumo dos componentes fundamentais encontrados em um computador pessoal e suas funções principais.

Placa-mãe

A placa-mãe é a plataforma que interliga todos os componentes (CPU, memória, armazenamento, placas de expansão) e fornece soquetes, barramentos e circuitos de alimentação.

Processador (CPU)

CPU (Central Processing Unit) é o 'cérebro' do computador: executa instruções, cálculos e controla o fluxo de dados entre os dispositivos.

Memória principal (RAM)

A RAM (Random Access Memory) armazena temporariamente dados e instruções em uso pelo processador, permitindo acesso rápido. É volátil (perde conteúdo sem energia).

Armazenamento (HDD / SSD)

Discos rígidos (HDD) e unidades de estado sólido (SSD) armazenam dados de forma persistente — sistema operacional, programas e arquivos do usuário.

O que é um Sistema Operacional?

Um sistema operacional é um software que gerencia os recursos de hardware e software de um computador, fornecendo uma interface entre o usuário e o hardware. Ele controla a execução de programas, gerencia memória, processa entrada e saída, e coordena dispositivos.

Exemplos de sistemas operacionais:

  • Windows
  • macOS
  • Linux
  • Android

O que é o Sistema de Arquivos?

O sistema de arquivos é uma estrutura que organiza e armazena dados em dispositivos de armazenamento, permitindo a criação, leitura, atualização e exclusão de arquivos e diretórios.

Como funciona em modo texto (cmd): No prompt de comando, comandos como dir, cd, mkdir são usados para navegar e gerenciar arquivos.

Como funciona em modo gráfico (explorer): No Explorador de Arquivos, você clica em ícones para abrir pastas, copiar arquivos, etc.

O que é Software de Escritório?

Software de escritório são programas utilizados para tarefas administrativas, como processamento de texto, planilhas, apresentações e gerenciamento de e-mails.

Microsoft Word

Aplicação: Processamento de texto para criar e editar documentos.

Microsoft Excel

Aplicação: Planilhas para cálculos, análise de dados e gráficos.

Microsoft PowerPoint

Aplicação: Criação de apresentações visuais com slides.
