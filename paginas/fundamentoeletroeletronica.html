Fundamentos da Eletroeletrônica

Introdução aos Fundamentos de Eletroeletrônica

A eletroeletrônica é a área que combina princípios da eletricidade e da eletrônica para o desenvolvimento de sistemas e dispositivos. Abaixo, uma lista de tópicos fundamentais:

  1. Conceitos básicos de eletricidade e circuitos.
  2. Leis fundamentais: Lei de Ohm, Lei de Kirchhoff.
  3. Componentes passivos: resistores, capacitores, indutores.
  4. Componentes ativos: diodos, transistores, circuitos integrados.
  5. Análise de circuitos elétricos.
  6. Fontes de alimentação e reguladores de tensão.
  7. Instrumentos de medição: multímetros, osciloscópios.
  8. Segurança elétrica e normas.
  9. Introdução à eletrônica digital.
  10. Prototipagem com placas de desenvolvimento.
  11. Aplicações práticas em sistemas automatizados.

Componentes Eletrônicos empregados em serviços

Resistor: Componente que limita a corrente elétrica. Diodo LED: Emite luz quando polarizado corretamente. Transistor: Amplifica sinais ou funciona como chave. Circuito Integrado: Conjunto de componentes miniaturizados em um chip.

Sensores Empregados em Sistemas Automatizados

Sensor de temperatura: Mede variações térmicas. Sensor de umidade: Detecta níveis de umidade no ar. Sensor de luminosidade: Avalia intensidade de luz. Outros incluem sensores de movimento, pressão, etc.

Arduino: O que é, Para que serve / aplicações, vantagens e desvantagens

Arduino é uma plataforma de prototipagem eletrônica open-source. Serve para criar projetos interativos, como robôs, sensores e automação. Aplicações incluem IoT, controle de motores e monitoramento ambiental. Vantagens: Fácil de usar, comunidade ativa, custo baixo. Desvantagens: Limitações de processamento para tarefas complexas.

Como o Arduino funciona? O que é eletrônica embarcada?

O Arduino funciona conectando componentes a pinos digitais e analógicos, programados via IDE. Eletrônica embarcada integra hardware e software em dispositivos dedicados, como em carros ou aparelhos médicos.

Arduino: qual a linguagem de programação usada

A linguagem usada é baseada em C/C++, com uma sintaxe simplificada para facilitar o desenvolvimento.

O que são entradas e saídas digitais em circuitos integrados como o Arduino

Entradas digitais leem sinais binários (0 ou 1). Saídas digitais enviam sinais binários para controlar dispositivos como LEDs ou relés.

O que são entradas e saídas analógicas em circuitos integrados como o Arduino

Entradas analógicas leem valores contínuos (0-1023). Saídas analógicas simulam sinais contínuos via PWM para controlar intensidade de luz ou velocidade de motores.

Calculadora de Potência Elétrica

Potência = tensão * corrente; ou Potência = corrente² * resistência; ou Potência = tensão² / resistência.

Exemplo em Arduino:

int sensor = analogRead(A0);
if(sensor > 500) {
    digitalWrite(13, HIGH);
} else {
    digitalWrite(13, LOW);
}

O que é um diodo? e Um Diodo LED? e um display 7 segmentos?

Diodo: Componente que permite corrente em uma direção. Diodo LED: Emite luz quando energizado. Display 7 segmentos: Mostra dígitos usando 7 LEDs em forma de segmentos.
