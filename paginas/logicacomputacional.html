O que são fluxogramas? Qual a sua simbologia básica?

Fluxogramas são representações gráficas de algoritmos, mostrando o fluxo de operações. Simbologia básica inclui: retângulo para processos, losango para decisões, oval para início/fim, seta para fluxo.

O que são algoritmos? Onde são usados?

Algoritmos são sequências finitas de passos para resolver problemas. Usados em programação, matemática, engenharia, etc.

O que é o portugol? O que é o Portugol Studio?

Portugol é uma linguagem de programação educacional em português. Portugol Studio é um ambiente de desenvolvimento para criar e testar algoritmos em Portugol.

O que são variáveis e constantes?

Variáveis armazenam valores que podem mudar. Constantes armazenam valores fixos.

Quais os tipos de dados mais usados em algoritmos (portugol)?

Booleanos (verdadeiro/falso), caracter (letra), cadeia (string), inteiro, real, vetor (array).
