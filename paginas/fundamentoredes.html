Fundamentos de Redes

O que são Redes de Computadores?

Redes conectam dispositivos para compartilhar recursos e informações.

O que é uma Rede mínima?

Dois dispositivos conectados diretamente.

O que é a internet? O que compõe?

Rede global de redes. Compõe: Servidores, roteadores, cabos, protocolos.

O que é web?

Sistema de documentos interligados via HTTP.

O que é o mapa de fibra óptica. Mostre o mapa global de fibra óptica da internet.

Mapa mostra cabos submarinos de fibra óptica conectando continentes.

Quais os componentes de hardware de uma rede wi-fi?

Roteador, antenas, dispositivos sem fio.

Quais os componentes de hardware de uma rede local com cabo de rede?

Switches, cabos Ethernet, NICs.

O que é um protocolo de comunicação?

Regras para transmissão de dados.

O que é o modelo OSI?

Modelo de 7 camadas para redes.

E o modelo TCP/IP?

Modelo de 4 camadas usado na internet.

O que é o protocolo IP V4 ? e o IP V6?

IPv4: 32 bits. IPv6: 128 bits, mais endereços.

Como é configurado o IP e o TCP em uma programação backend com node.js?

Usando módulos como 'net' ou 'http'. Exemplo:

const net = require('net');
const server = net.createServer((socket) => {
    socket.write('Hello from TCP server!');
    socket.end();
});
server.listen(8080);

O que é o protocolo HTTP?

Protocolo para web, transfere hipertexto.

E o HTTPS ?

HTTP seguro com criptografia SSL/TLS.

Cite exemplo de um framework para programação do protocolo HTTP com uso do node.js?

Express.js.

Apresente 3 exemplos de código Backend para troca de mensagens entre front e backend.

1. GET request:

app.get('/api', (req, res) => res.send('Hello'));

2. POST request:

app.post('/data', (req, res) => res.json(req.body));

3. WebSocket para mensagens em tempo real:

const WebSocket = require('ws');
const wss = new WebSocket.Server({ port: 8080 });
wss.on('connection', (ws) => {
    ws.on('message', (message) => {
        ws.send('Received: ' + message);
    });
});
